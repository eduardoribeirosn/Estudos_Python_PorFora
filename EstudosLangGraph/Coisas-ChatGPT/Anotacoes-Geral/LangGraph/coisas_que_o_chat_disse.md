# LangGraph

LLM -> Large Language Model

Exemplos {
    GPT-4
    Claude
    Gemini
}

Esses modelos consegue:
- responder perguntas
- escrever códigos
- resumir textos
- traduzir
- analisar dados textuais

------

##### Node
Um node é uma função

##### Edge
Um edge é uma conexão entre nodes

-----

State (estado)
é o dicionário compartilhado entre nodes.
(cada node poder [ler e alterar])
